# Atur Pabrik

Atur Pabrik adalah sistem operasional manufaktur untuk mengelola stok, memantau produksi secara real-time, dan menyusun planning berbasis kondisi material aktual. Dibangun untuk pabrik yang butuh alur kerja lebih sederhana, cepat, dan relevan dengan proses lapangan.

## Fitur Utama

- Manajemen stok gudang untuk raw material, finish good, dan return
- Pencatatan barang masuk, keluar, transfer, dan penyesuaian stok
- Monitoring produksi berjalan per mesin atau lane
- Update output produksi real-time melalui aplikasi dan API
- Tracking target qty, waktu target, hasil aktual, dan persentase capaian
- Planning produksi harian, mingguan, hingga bulanan
- Perhitungan material berdasarkan stok existing, rework, return, dan crush material

## Tujuan

Atur Pabrik dirancang untuk membantu pabrik memiliki visibilitas operasional yang lebih cepat dan akurat, mulai dari gudang hingga produksi, tanpa alur kerja yang rumit.

## Tech Stack

- Laravel
- REST API
- Web dashboard
- Mobile app integration
